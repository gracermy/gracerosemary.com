How to Play

The 9×9 Grid

The board is a 9×9 grid split into nine 3×3 boxes. Some numbers are already placed — these are your clues.

One Simple Rule

Every row, every column, and every 3×3 box must contain the numbers 1 through 9 exactly once. No repeats.

See the Constraints

Tap any cell and its row, column, and box light up. This shows you which numbers are already taken in that zone.

Use Elimination

If a row already has 1, 3, 4, 5, 6, 7, 8, 9 — the missing number must be 2. Start with nearly-complete groups.

Track Numbers

When you select a cell with a number, all cells with that same number glow. This helps you spot patterns and gaps.

Mistakes? No Problem

Wrong answers turn pink when you check. Fix the mistake with the right number and submit it again!
